# 📱 SNS日本語マスター

SNS上の日本語表現を学ぶための、インタラクティブなウェブアプリケーションです。日本語学習者向けに、実際のSNS投稿風のUIで、効果的に学習できます。

## ✨ 主な機能

### 🎯 リアルなSNS体験
- **Twitter/X、Instagram、LINE風のUI**で実際のSNS投稿を再現
- 各プラットフォームに特化したデザイン

### 📚 効果的な学習機能
- **インタラクティブな解説**: 投稿内の表現をクリックすると詳細説明が表示
- **詳細な表現解説**: 
  - 読み方（ふりがな）
  - 英語の意味
  - 詳しい解説
  - フォーマル度
  - 使用例
  - 注意点
  - 類似表現・反対表現
  - 音声読み上げ機能

### 🎮 クイズモード
- **選択問題**と**記述問題**
- リアルタイムの正解率表示
- 詳しい解説付き

### 🔍 強力な検索・フィルター
- キーワード検索
- JLPTレベル別フィルター（N5-N1）
- カテゴリー別フィルター
  - 若者言葉
  - ネットスラング
  - 略語
  - 絵文字・顔文字
  - 挨拶
  - リアクション
  - 感情表現
  - トレンド
  - 日常会話
- プラットフォーム別フィルター

### 📊 学習進捗管理
- 学習済み投稿のマーク
- お気に入り機能
- LocalStorageで進捗を自動保存

### 📱 レスポンシブデザイン
- スマートフォン、タブレット、PCに完全対応
- 美しいグラデーションとモダンなUI

## 🛠 技術スタック

## 🚀 セットアップ

### 必要な環境
- Node.js 16以上
- npm または yarn

### インストール

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ブラウザで開く
# http://localhost:5173
```

### ビルド

```bash
# 本番用ビルド
npm run build

# プレビュー
npm run preview
```

## 📖 使い方

### フィードモード
1. トップページで実際のSNS投稿風の教材を閲覧
2. 投稿内の**ハイライトされた表現**をクリックして詳細を確認
3. 学習ポイントセクションの表現をクリックして詳しい解説を表示
4. **学習済みにする**ボタンで進捗を記録
5. **ブックマークアイコン**でお気に入りに追加

### フィルター機能
1. 検索バーでキーワード検索
2. **フィルター**ボタンをクリック
3. JLPTレベル、カテゴリー、プラットフォームを選択
4. **すべてクリア**で初期状態に戻る

### クイズモード
1. **クイズ**タブをクリック
2. 問題に回答
3. **回答する**ボタンで採点
4. 解説を読んで理解を深める
5. **次の問題**で続ける

### お気に入り
1. **お気に入り**タブで保存した投稿を一覧表示
2. 復習に最適

## 🎨 デザインの特徴

- **グラデーション背景**: 目に優しい配色
- **プラットフォーム別カラースキーム**: 
  - Twitter/X: ブルー系
  - Instagram: ピンク/パープル系
  - LINE: グリーン系
- **アニメーション**: スムーズなトランジション
- **カード型レイアウト**: 現代的なデザイン

## 📚 収録内容

現在10個のSNS投稿教材と5つのクイズを収録：

- マジで、やばい、エモい（若者言葉）
- おつかれ、サクッと（日常会話）
- それな、わかりみが深い（リアクション）
- 映える、めちゃくちゃ（トレンド）
- おけまる、りょ（略語）
- バズる、えぐい、ｗ（ネットスラング）
- 尊い、推し、ガチ恋（感情表現）
- なう、詰む、無理ゲー（ネットスラング）
- チル、癒やされる（感情表現）
